## 4. 后端 S7-S8：自动循环编排

| 功能 | 说明 |
|------|------|
| **orchestration/status** | GET /api/tasks/{id}/orchestration/status |
| **orchestration/step** | POST /api/tasks/{id}/orchestration/step |
| **orchestration/run** | POST /api/tasks/{id}/orchestration/run |

### 自动循环规则

| 状态 | 自动动作 | 停止原因 |
|------|---------|----------|
| draft | generate-ticket → ticket_ready | — |
| ticket_ready | dispatch → dispatched | — |
| dispatched + 无 run | create+start AgentRun | waiting_for_agent_result |
| dispatched + running | — | waiting_for_agent_result |
| dispatched + succeeded | submit_result + 创建 artifact | — |
| dispatched + failed | — | agent_failed |
| result_submitted | start-review → reviewing | — |
| reviewing + 低风险 decision | auto-approve → approved | — |
| reviewing + 无 decision | evaluate → human_required | human_required |
| human_required | — | human_required |
| approved | — | approved_waiting_archive |

### 安全限制

- 不创建重复 AgentRun（running 时不创建第二个）
- failed/canceled 不再创建新 run
- 跨 task 数据隔离（所有 query 带 task_id）
