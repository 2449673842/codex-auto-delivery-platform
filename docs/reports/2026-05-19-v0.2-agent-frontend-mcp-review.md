## 5. 是否使用 Playwright MCP 真实浏览器操作

是。使用 `playwright` Node.js API 启动 Chromium 浏览器，模拟真实用户操作：

- 点击按钮
- 填写表单
- 选择下拉选项
- 勾选复选框
- 提交表单
- 观察页面 DOM 变化
- 监控 console error 和 network 失败

未使用 curl 或 API 代替浏览器操作。
