# v0.4 S10 — Prompt Template Preview API

**Date**: 2026-05-21  
**Author**: AI1  
**Status**: Completed

## Summary

Added `POST /api/prompt-templates/preview` for generating per-mode system/user prompt previews. Reuses S9 AI Context Packet data (via `ai_context_packet_service.preview()`) to build structured prompts with safety boundaries and token budget estimation.

## Scope

- Stateless preview only — no DB writes, no real AI calls, no file scanning
- 5 mode templates: planning, patch_generation, review, risk, browser_reviewer
- System prompt includes mode-specific output constraints and safety boundaries
- User prompt includes task context matched via Context Selector
- SHA256 hashes (system_prompt_hash, user_prompt_hash, prompt_hash)
- Token budget estimation (ceil(char_count/4))

## API

`POST /api/prompt-templates/preview`

Input: `{task_goal, module_name, task_type, mode}`  
Output: `{template_id, mode, system_prompt_preview, user_prompt_preview, output_contract, context_packet_hash, prompt_hash, token_budget, redaction_applied, warnings}`
